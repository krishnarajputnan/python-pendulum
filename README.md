# Python Pendulum Simulation

A programmatic model and simulation of deterministic pendulum kinematics built in Python. 

## Overview
This project visualizes the motion of a simple pendulum by calculating its angle and velocity over time. It demonstrates the translation of physical laws and differential equations into functional code.

The core motion relies on the standard pendulum equation:
$$ \frac{d^2\theta}{dt^2} + \frac{g}{L}\sin\theta = 0 $$

## Technologies Used
* **Python 3**

## How to Run
To run this simulation locally, download the code and execute the main script in your terminal:
